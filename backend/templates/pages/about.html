About Pong

Pong is a web-game we made based on Pong (1972). It is also the final project for the 42 core curriculum.

The Pong website offers classic 1v1, 2v2, and tournament-style gameplay, along with modern features like user authentication, matchmaking, and game customization. Players can also enjoy an additional Tic-Tac-Toe game.

Built as a single-page application (SPA) using Django and other web technologies, Pong a smooth user experience and dynamic content loading, showcasing the skills acquired throughout the 42 program.
